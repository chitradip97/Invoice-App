# POS Billing System with core PHP
## Features of the Project

#### I am Using Core Php, Javascript and MYSQL to built the project.
The POS(Point Of Sale) system offers comprehensive features for businesses, with distinct roles for Admin and User. Its dynamic dashboard provides insights into sales metrics, and users can manage customers, products, and categories easily. Orders can be generated seamlessly, and dynamic invoice generation streamlines transactions. Technologies include Core PHP, JavaScript, MySQL, HTML, CSS, Bootstrap, Ajax, and jQuery, ensuring a robust and responsive application.
